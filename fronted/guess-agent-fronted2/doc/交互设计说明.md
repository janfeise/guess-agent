## 📍 交互分类总览

### 交互类型分布

| 类型         | 数量 | 示例                       |
| ------------ | ---- | -------------------------- |

## 🏠 首页 (pages/index/index.vue)

### 交互1: 单词输入

**触发**：用户在文本框中输入

**行为**：

```javascript
@input="handleInputChange"
  ↓
gameForm.startWord = e.detail.value
// 实时更新表单数据
```

**验证**：

- ✅ 非空检查
- ✅ 长度限制 (2-50字)
- ✅ 特殊字符过滤

**反馈**：按钮状态 (disabled/enabled)

---

### 交互2: 难度选择

**触发**：点击难度按钮

**行为**：

```javascript
@click="gameForm.difficulty = key"
  ↓
修改gameForm.difficulty
触发UI更新（active状态切换）
```

**视觉反馈**：

- 选中: 白色背景 + 主色文字
- 未选: 灰色背景 + 灰色文字

**限制**：单选，一次只能选一个

---

### 交互3: 开始游戏

**触发**：点击「开始游戏 🚀」按钮

**流程**：

```
表单验证
  ├─ :disabled={!gameForm.startWord}  (按钮禁用状态)
  ├─ @click="handleStartGame"
  └─ if (!gameForm.startWord.trim()) {
      uni.showToast('请输入初始单词')
      return
    }
    ↓
gameLoading.value = true (UI转为加载中)
    ↓
clearCurrentGame()  (清除前一次游戏)
    ↓
newGame = createNewGame(startWord, difficulty)
    ↓
uni.showToast(...success...)
    ↓
setTimeout(() => {
  uni.navigateTo('/pages/gamePlay/gamePlay')
    ↓
  调用 gamePlayPage.initGame(newGame)  (传递游戏数据)
}, 1000)
    ↓
gameLoading.value = false (UI恢复)
```

**反馈**：

- ✅ 按钮 disabled 状态切换
- ✅ 加载中文案变化
- ✅ Toast 成功提示
- ✅ 页面渐进式导航

**错误处理**：

- ❌ 空输入 → Toast 错误消息
- ❌ 异常 → Toast 失败消息 + 保持在当前页

## 🎮 游戏进行页 (pages/gamePlay/gamePlay.vue)

### 交互1: 用户聊天输入

**触发**：用户在输入框中输入并点击提交

**流程**：

```
用户输入 "它是生物吗?"
    ↓
@focus="handleInputFocus"  (自动滚动到底部)
    ↓
@click="handleSubmitMessage"
    ├─ 输入验证
    │  ├─ !trimmedInput → Toast('请输入内容')
    │  └─ return
    │
    ├─ 输入类型判断
    │  ├─ detectInputType() → 'question'
    │  └─ 或 'guess'
    │
    ├─ 限制检查
    │  ├─ 若是问题:
    │  │   questionsUsed >= questionLimit
    │  │   → Toast('提问次数已用完')  + return
    │  │
    │  └─ 若是猜测:
    │      guessesUsed >= guessLimit
    │      → Toast('猜测次数已用完')  + return
    │
    ├─ 添加用户消息
    │  └─ messages.push({ type: 'user', text })
    │
    ├─ 更新计数器
    │  ├─ if (问题): questionsUsed++
    │  └─ if (猜测): guessesUsed++
    │
    ├─ 清空输入框
    │  └─ userInput.value = ''
    │
    ├─ 显示加载状态
    │  └─ isLoading.value = true
    │     (显示三个点动画)
    │
    ├─ 模拟AI思考延迟
    │  └─ await delay(800-2000ms)
    │
    ├─ 生成AI响应
    │  └─ aiResponse = generateAIResponse(...)
    │
    ├─ 添加AI消息
    │  └─ messages.push({ type: 'agent', text: aiResponse })
    │
    ├─ 更新轮次
    │  └─ currentGame.roundNum++
    │
    ├─ 判断是否显示Hunch提示
    │  ├─ if (questionsUsed % 3 === 0 && > 0):
    │  │   showHunchCard = true
    │  │   hunchHint = generateHunchHint(startWord)
    │  │
    │  │   await delay(500)  (先等待消息渲染)
    │  │
    │  │   setTimeout(() => {
    │  │     showHunchCard = false
    │  │   }, 4000)  (显示3秒后隐藏)
    │
    ├─ 游戏结束判断 (如果是猜测)
    │  ├─ 猜对:
    │  │   messages.push('恭喜！你猜对了！🎉')
    │  │   status = 'won'
    │  │   result = createGameResult(..., true)
    │  │   saveGameResult(result)
    │  │   setTimeout(() => navigateToResult(), 1500)
    │  │
    │  ├─ 猜错 & 还有机会:
    │  │   messages.push('不对呢。再试试？')
    │  │
    │  └─ 猜错 & 无机会:
    │      status = 'lost'
    │      result = createGameResult(..., false)
    │      saveGameResult(result)
    │      setTimeout(() => navigateToResult(), 1500)
    │
    ├─ 保存游戏状态
    │  └─ saveCurrentGame(currentGame)
    │
    ├─ 关闭加载状态
    │  └─ isLoading.value = false
    │
    └─ 重新聚焦输入框
       └─ nextTick(() => handleInputFocus())
```

**反馈系统**：

| 阶段     | 反馈            | 效果        |
| -------- | --------------- | ----------- |
| 输入验证 | Toast           | 2秒消失     |
| 限制检查 | Toast           | 2秒消失     |
| AI思考   | Loading动画     | 3个点跳跃   |
| 新消息   | 列表渲染 + 滚动 | 自动到底部  |
| Hunch    | 卡片显示        | 3秒后隐藏   |
| 游戏结束 | 消息 + 延迟跳转 | 1.5秒后转移 |

### 交互2: 实时自动滚动

**触发**：新消息添加 or 输入框获焦

**行为**：

```javascript
handleInputFocus() {
  nextTick(() => {
    uni.pageScrollTo({
      scrollTop: 10000,    // 大数值确保到底
      duration: 300
    })
  })
}
```

**效果**：平滑滚动到底部，3秒内完成

## 📋 历史记录页 (pages/history/history.vue)

### 交互1: 筛选切换

**触发**：点击筛选标签

**标签**：

- 全部 (all)
- 获胜 (won)
- 落败 (lost)

**行为**：

```javascript
<FilterTab
  v-model="activeFilter"
  @change="handleFilterChange"
/>
    ↓
activeFilter被更新
    ↓
filteredGames计算属性重新计算
    ↓
列表重新渲染
```

**过滤逻辑**：

```javascript
if (activeFilter === "all") return allGames;
else if (activeFilter === "won") return allGames.filter((g) => g.isWon);
else if (activeFilter === "lost") return allGames.filter((g) => !g.isWon);
```

**视觉反馈**：

- 选中标签: 主色背景 + 白色文字
- 未选: 灰色背景

### 交互3: Toast 反馈

**使用场景**：

```
设置初始单词 → uni.showToast('请输入初始单词')
提问次数用完 → uni.showToast('提问次数已用完')
游戏开始成功 → uni.showToast('游戏开始！', icon: 'success')
```

**配置**：

```javascript
uni.showToast({
  title: "文案",
  icon: "none" | "success" | "loading",
  duration: 2000, // 毫秒
  mask: false,
});
```

---

### 交互4: 本地存储

**自动进行**（用户无感知）：

```
游戏过程中 → saveCurrentGame() 每次状态更新
游戏结束 → saveGameResult() 保存到历史
首页加载 → getGameHistory() 显示最近记录
历史页面 → getGameHistory() + 筛选
进入任何页面 → onMounted() 加载必要数据
```
