## 🎮 游戏玩法

### 难度配置

| 难度 | 提问名额 | 猜测次数 |
| ---- | -------- | -------- |
| 简单 | 15       | 5        |
| 普通 | 10       | 3        |
| 困难 | 7        | 2        |

### 游戏流程

1. **输入单词** - 在首页输入起始词
2. **选择难度** - 选择游戏难度等级
3. **开始游戏** - 跳转到gamePlay页面
4. **提问AI** - 问概念性问题（每问消耗1名额）
5. **获得提示** - 每3次提问后获得Hunch提示
6. **猜测答案** - 输入目标词（每次消耗1猜测次数）
7. **查看结果** - 展示成绩和统计

## 🐛 常见问题

**Q: 如何重置游戏数据？**

```javascript
import { clearAllData } from "@/utils/storage";
clearAllData(); // 清除所有本地数据
```

**Q: 如何自定义气泡颜色？**
修改 `components/ChatBubble.vue` 的样式中的颜色值

**Q: 如何增加更多难度？**
在 `utils/gameLogic.js` 添加新等级到 `DIFFICULTY_LEVELS`

**Q: 游戏历史如何导出？**
