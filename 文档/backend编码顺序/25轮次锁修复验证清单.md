## 集成测试场景

### 🧪 场景 1: 用户提问 → 系统提问

```
【用户请求】
POST /games/{game_id}/turns
{
  "content": "是理科吗?"
}

【预期响应】
{
  "status": 200,
  "body": {
    "phase": "awaiting_answer",           ← ✅ 关键：不是 user_turn
    "system_question": "是物理吗？",       ← ✅ 新增字段
    "message": "我的下一个问题是：是物理吗？",
    "history": [
      {
        "actor": "user",
        "turn_type": "question",
        "input_text": "是理科吗?"
      },
      {
        "actor": "system",
        "turn_type": "question",
```

### ✅ 关键日志行

```
1. INFO: 收到回合请求: game_id=xxx turn_type=input
   └─ 用户提交输入

2. INFO: LLM 请求 [answer_question]:
   └─ 系统回答用户问题

3. INFO: LLM 原始返回 [answer_question/parsed]: { answer: "yes", ... }
   └─ 系统回答完成

4. INFO: LLM 请求 [decide_agent_action]:             ← ✅ 关键！新增
   └─ 系统进行决策

5. INFO: LLM 原始返回 [decide_agent_action/parsed]: { action: "question", ... }
   └─ 系统决策完成

6. INFO: HTTP Request: POST /games/{game_id}
   └─ 阶段转移已保存

7. INFO:     172.18.0.1:xxxxx - "POST /api/v1/games/.../turns HTTP/1.1" 200 OK
   └─ API 请求完成
```

**关键验证**：

- [ ] 每个用户输入都触发了 `decide_agent_action`
- [ ] `decide_agent_action` 返回的 action 是 "question" 或 "guess"
