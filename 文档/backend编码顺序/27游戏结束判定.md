## 问题诊断

### 现象
用户成功猜出系统词后，游戏仍处于 `awaiting_answer` 或 `user_turn` 阶段，系统继续提问而不是结束游戏。

### 根本原因分析

**原因 1: Intent 识别失败**
- 用户输入 "我猜是艺术史" → `parse_user_intent()` 应返回 `intent: "guess"`
- 但实际可能返回 `intent: "answer"` 或 `intent: "invalid"`
- 导致用户的猜测没有进入 `_handle_user_guess()` 流程

**原因 2: 游戏阶段锁定**
- 当前阶段为 `awaiting_answer`（等待回答系统提问）
- 用户输入猜测但仍被识别为 "answer"（因为 intent 错误）
- 即使识别为 "guess"，也会被 `_is_turn_allowed()` 拒绝（awaiting_answer 阶段不允许 guess）

**原因 3: 猜测验证缺失**
- 即使进入 `_handle_user_guess()`，如果用户猜测与系统词相同
- 缺少对"系统词"的正确解密和比对

**原因 4: 系统猜测判定缺失**
- 用户需要判定系统猜测是否正确（awaiting_judgement 阶段）
- 如果判定正确，系统赢且游戏结束
- 但当前代码可能没有正确处理这个场景

## 游戏结束的三种场景

### 场景 1: 用户猜对系统词（用户赢）

**流程图:**
```
user_turn 
  ↓ 用户输入 "我猜是艺术史"
  ↓ parse_user_intent() → intent: "guess"
  ↓ submit_turn() → _handle_user_guess()
  ↓ 比对: user_guess vs system_word → 相等
  ↓ 调用 finish_game(reason: "user_win")
  ↓ 返回 status: "finished", result: "user_win"
```

**涉及的函数:**
- `parse_user_intent()` - 识别 guess intent
- `_handle_user_guess()` - 处理用户猜测
- `finish_game()` - 结束游戏

### 场景 2: 系统猜对用户词（系统赢）

**流程图:**
```
awaiting_judgement
  ↓ 系统已提出猜测: "是不是艺术史？"
  ↓ 用户输入 "是的" / "对的"
  ↓ parse_user_intent() → intent: "judge" 或 "answer"（取决于识别）
  ↓ submit_turn() → _handle_agent_guess_judgement()
  ↓ 判定: 用户确认为正确
  ↓ 调用 finish_game(reason: "agent_win_guessed_correctly")
  ↓ 返回 status: "finished", result: "agent_win"
```

**涉及的函数:**
- `_handle_agent_guess_judgement()` - 处理用户对系统猜测的判定
- `finish_game()` - 结束游戏

### 场景 3: 达到回合上限（游戏超时）

**流程图:**
```
无论哪个阶段
  ↓ 检查 round_count >= max_rounds（通常为 20）
  ↓ submit_turn() 返回 "达到最大轮数，游戏结束"
  ↓ 调用 finish_game(reason: "max_rounds_reached")
  ↓ 返回 status: "finished", result: "draw"
```

**涉及的函数:**
- `submit_turn()` - 提前检查轮数
- `finish_game()` - 结束游戏

## 核心实现策略

### 策略 1: Intent 识别强化

用户的猜测表达方式多样：
- "我猜是艺术史"
- "猜是艺术史"
- "应该是艺术史"
- "可能是艺术史"
- "就是艺术史吧"

需要在 `_infer_local_intent()` 或 LLM 的 `turn_router.txt` 中增强识别：

```python

    normalized = cls._normalize_intent_text(user_input)
    
    # 增强猜测识别

        guess_word = cls._extract_guess_word(normalized, guess_markers)

            "intent": "guess",
            "guess_word": guess_word,
            # ...
        }
```

```python
async def submit_turn(self, game_id: str, question: str, mode: str = "agent"):
    # 1️⃣ 获取游戏并检查基本状态
    game = await self.repository.get_game(game_id)
    if not game:
        raise ValueError(f"Game {game_id} not found")
    if game.get("status") == "finished":
        raise ValueError(f"Game {game_id} has finished")
    
    # 2️⃣ 检查轮数上限（提前结束）
    max_rounds = 20
    if game.get("round_count", 0) >= max_rounds:
        now = datetime.now()
        await self.repository.finish_game(game_id, "max_rounds_reached", now)
        return {
            "game_id": game_id,
            "status": "finished",
            "result": "draw",
            "message": f"已达最大轮数({max_rounds})，游戏结束",
        }
    
    # 3️⃣ 解析用户意图
    metadata = game.get("metadata", {}) or {}
    phase = metadata.get("phase", "user_turn")
    history = game.get("history", [])
    
    intent_result = await self.agent.parse_user_intent(question, history)
    intent = intent_result.get("intent", "invalid")
    
    # 4️⃣ 轮次锁校验
    if not self._is_turn_allowed(phase, intent):
        return self._reject_turn(game_id, phase, 
                                f"当前阶段({phase})不允许此操作({intent})", history)
    
    # 5️⃣ 优先处理用户猜测（任何阶段都可以猜测）
    if intent == "guess":
        return await self._handle_user_guess(game, intent_result)
    
    # 6️⃣ 处理其他意图（根据阶段）
    if phase == "awaiting_judgement":
        return await self._handle_agent_guess_judgement(game, question)
    
    if phase == "awaiting_answer":
        return await self._handle_user_answer(game, question, history, intent_result)
    
    if phase == "user_turn":
        if intent == "invalid":
            return self._reject_turn(game_id, phase, "请输入有效问题或猜测", history)
        if intent == "question":
            return await self._handle_user_question(game, question, history, mode)
        if intent == "answer":
            return self._reject_turn(game_id, phase, "现在轮到你提问或猜词", history)
    
    return self._reject_turn(game_id, phase, "游戏状态异常", history)
```

### 实现 4: 完整的用户猜测处理

**文件:** `backend/app/services/game_service.py`

```python
async def _handle_user_guess(self, game: dict, intent_result: dict):
    """
    处理用户猜测
    
    流程:
    1. 提取猜测词汇
    2. 解密系统词
    3. 对比验证
    4. 判定结果并结束游戏或继续
    """
    game_id = game["game_id"]
    user_guess = intent_result.get("guess_word", "").strip()
    current_phase = game.get("metadata", {}).get("phase", "user_turn")
    
    # ❌ 验证猜测词不为空
    if not user_guess:
        return self._reject_turn(game_id, current_phase,
                                "猜测词不能为空，请输入有效猜测", 
                                game.get("history", []))
    
    # 🔓 解密系统词
    try:
        target_word = self.reveal_target_word(game, self.settings)
    except Exception as e:
        logger.error("解密系统词失败 [game_id=%s]: %s", game_id, e)
        return self._reject_turn(game_id, current_phase,
                                "系统词验证失败，请稍后重试",
                                game.get("history", []))
    
    # ✅ 猜测正确 - 游戏结束（用户赢）
    if self.is_guess_correct(user_guess, target_word):
        now = datetime.now()
        await self.repository.finish_game(game_id, "user_win", now)
        
        # 记录用户的猜测到 history
        final_history = game.get("history", []) + [
            self._make_history_item(
                round_no=len(game.get("history", [])) + 1,
                actor="user",
                turn_type="guess",
                input_text=user_guess,
                answer_text="correct_guess",
                result="user_guessed_correctly",
                source_word_owner="system",
                visible_to_agent=False,
                visible_to_player=True,
            )
        ]
        
        return {
            "game_id": game_id,
            "status": "finished",
            "result": "user_win",
            "message": f"🎉 恭喜你猜对了！目标词是：{target_word}",
            "target_word": target_word,
            "guess": user_guess,
            "history": final_history,
        }
    
    # ❌ 猜测错误 - 继续游戏
    else:
        updated_history = game.get("history", []) + [
            self._make_history_item(
                round_no=len(game.get("history", [])) + 1,
                actor="user",
                turn_type="guess",
                input_text=user_guess,
                answer_text="wrong_guess",
                result="user_guessed_incorrectly",
                source_word_owner="system",
                visible_to_agent=False,
                visible_to_player=True,
            )
        ]
        
        await self.repository.update_game_state(
            game_id,
            history=updated_history,
            summary=self.memory_policy.build_summary(updated_history),
            round_count=len(updated_history),
            phase="user_turn",
            awaiting_judgement=False,
            awaiting_answer=False,
            pending_guess="",
            pending_question="",
            agent_confidence=0.0,
            last_actor="user",
            last_intent="guess",
            game_mode=game.get("metadata", {}).get("game_mode"),
            reasoning_scope=game.get("metadata", {}).get("reasoning_scope"),
            next_actor="user",
            expected_turn_type="guess",  # 提示用户可继续猜测
        )
        
        return {
            "game_id": game_id,
            "status": game.get("status", "active"),
            "phase": "user_turn",
            "result": "guess_incorrect",
            "message": f"❌ 很遗憾，'{user_guess}' 不是目标词。游戏继续，继续提问或猜测吧！",
            "guess": user_guess,
            "target_word_hint": target_word[:1] + "*" * (len(target_word) - 1),  # 提示首字
            "history": updated_history,
        }
```

```python
    处理用户对系统猜测的判定
    
    流程:
    1. 提取用户的判定（是/否）
    2. 如果是，系统赢，游戏结束
    3. 如果否，继续游戏

    pending_guess = game.get("metadata", {}).get("pending_guess", "")

        "是", "正确", "对", "没错", "对了", "对的", 
        "是的", "yes", "y", "true", "确实", "同意"
    ]
    
    if any(judgment_text == p for p in affirmative_patterns) or \
       any(judgment_text.startswith(p) for p in affirmative_patterns[:5]):
        
        now = datetime.now()
        await self.repository.finish_game(
```

## 常见陷阱

### 陷阱 1: Intent 识别复杂性

**问题:**
- 用户输入 "应该是计算机" 可能被识别为 "answer" 而不是 "guess"
- 需要 LLM 和本地启发式的双重识别

**解决:**
```python
# LLM 识别失败时，本地启发式补救
local_intent = self._infer_local_intent(user_input)
if local_intent is not None and parsed.get("intent") == "invalid":
    parsed.update(local_intent)
```

### 陷阱 2: 阶段锁定太严格

**问题:**
- `awaiting_answer` 阶段拒绝了用户的猜测（即使意图正确）
- `awaiting_judgement` 阶段也拒绝了用户的猜测

**解决:**
```python
# 任何阶段都允许猜测
if intent == "guess":
    return phase != "finished"  # 只有结束游戏拒绝
```

### 陷阱 3: 解密失败处理不完善

**问题:**
- 系统词解密异常，导致猜测验证失败
- 用户体验不佳

**解决:**
```python
try:
    target_word = self.reveal_target_word(game, self.settings)
except Exception as e:
    logger.error("解密失败: %s", e)
    return self._reject_turn(game_id, phase,
                            "系统词验证失败，请稍后重试",
                            game.get("history", []))
```

### 陷阱 4: 猜测词标准化不一致

**问题:**
- 用户输入 "计算 机" vs 系统词 "计算机"
- 用户输入 "计算机 " vs 系统词 "计算机"
- 大小写不同

**解决:**
```python
@staticmethod
def is_guess_correct(user_guess: str, target_word: str) -> bool:
    # 标准化处理
    user_normalized = user_guess.strip().lower()
    target_normalized = target_word.strip().lower()
    # 移除所有空格
    user_normalized = user_normalized.replace(" ", "")
    target_normalized = target_normalized.replace(" ", "")
    return user_normalized == target_normalized
```

### 陷阱 5: History 记录不完整

**问题:**
- 用户猜测会被重复计数
- 系统决策没有正确记录

**解决:**
```python
# 每次用户操作都创建 history item
self._make_history_item(
    round_no=len(game.get("history", [])) + 1,  # 正确计数
    actor="user",
    turn_type="guess",
    input_text=user_guess,
    answer_text="correct_guess" or "wrong_guess",
    visible_to_player=True,
)
```

### 陷阱 6: API 响应格式不统一

**问题:**
- 有些情况返回 `target_word`，有些不返回
- 客户端逻辑混乱

**解决:**
```python
# 游戏结束时统一返回这些字段
finished_response = {
    "game_id": game_id,
    "status": "finished",
    "result": "user_win" or "agent_win" or "draw",
    "message": "...",
    "history": sanitized_history,
    "target_word": target_word if result == "user_win" else None,
    "agent_guess": agent_guess if result == "agent_win" else None,
}
```
