# 轮次锁修复对比：修复前 vs 修复后

## 一览表

| 方面             | ❌ 修复前                         | ✅ 修复后                                        |
| ---------------- | --------------------------------- | ------------------------------------------------ |
| **用户提问后**   | 返回 user_turn                    | 系统决策 → awaiting_answer 或 awaiting_judgement |
| **用户回答后**   | 直接返回 user_turn                | 系统决策 → awaiting_answer 或 awaiting_judgement |
| **游戏循环**     | 用户→系统(回答)→用户→...          | 用户→系统(回答+决策)→用户→...                    |
| **系统行为**     | 被动回答                          | 能动决策                                         |
| **API phase 值** | user_turn → user_turn → user_turn | user_turn → awaiting_answer ⇄ user_turn          |

## 代码对比

### `_handle_user_question()` 对比

#### ❌ 修复前（错误）

```python
async def _handle_user_question(self, game: dict, question: str, history: list[dict], mode: str = "agent"):
    # 1. 系统回答用户的问题
    answer_result = await self.agent.answer_question(question, target_word)

    # 2. 建立 history
    raw_history = history + [
        # 记录用户问题和系统回答
        self._make_history_item(...)
    ]

    # 3. 直接返回 user_turn（错误！系统不进行决策）
    await self.repository.update_game_state(
        game["game_id"],
        history=raw_history,
        phase="user_turn",  # ← 直接回到用户轮（错误）
        next_actor="user",  # ← 继续轮到用户
        expected_turn_type="question",  # ← 期望用户继续提问
    )

    return {
        "phase": "user_turn",  # ← 用户可以继续提问
        "message": "回答完毕，现在轮到你继续提问或直接猜答案。"
    }
```

**问题**：

- 系统永远不进行决策
- 用户可以无限提问
- 系统没机会猜测
- 游戏变成"用户提问环"

#### ✅ 修复后（正确）

```python
async def _handle_user_question(self, game: dict, question: str, history: list[dict], mode: str = "agent"):
    # 1. 系统回答用户的问题
    answer_result = await self.agent.answer_question(question, target_word)

    # 2. 建立 history（只包含用户问题和系统回答）
    raw_history = history + [
        self._make_history_item(...)
    ]

    # 3. 系统进行决策（新增！）
    summary = self.memory_policy.build_summary(raw_history)
    agent_decision = await self.agent.decide_agent_action(raw_history, summary)
    agent_action = agent_decision.get("action", "question")  # "question" 或 "guess"

    updated_history = raw_history

    # 4. 根据决策条件路由
    if agent_action == "question":
        # 系统决定继续提问
        agent_question = agent_decision.get("question", "请继续提供更多信息。")
        updated_history = updated_history + [
            # 添加系统的新问题到 history
            self._make_history_item(
                actor="system",
                turn_type="question",
                input_text=agent_question,
            )
        ]

        # 更新状态为 awaiting_answer（等待用户回答系统的新问题）
        await self.repository.update_game_state(
            game["game_id"],
            history=updated_history,
            phase="awaiting_answer",  # ← 等待用户回答系统的问题
            next_actor="user",
            expected_turn_type="answer",  # ← 期望用户回答，不是提问
        )

        return {
            "phase": "awaiting_answer",  # ← 用户现在应该回答
            "system_question": agent_question,  # ← 显示系统的下一个问题
            "message": f"我的下一个问题是：{agent_question}"
        }

    else:  # agent_action == "guess"
        # 系统决定开始猜测
        guess_word = agent_decision.get("guess_word", "unknown")
        updated_history = updated_history + [
            # 添加系统的猜测到 history
            self._make_history_item(
                actor="system",
                turn_type="guess",
                input_text=guess_word,
            )
        ]

        # 更新状态为 awaiting_judgement（等待用户判断猜测）
        await self.repository.update_game_state(
            game["game_id"],
            history=updated_history,
            phase="awaiting_judgement",  # ← 等待用户判断
            next_actor="user",
            expected_turn_type="judge",  # ← 期望用户判断对错
        )

        return {
            "phase": "awaiting_judgement",  # ← 用户现在应该判断
            "system_guess": guess_word,  # ← 显示系统的猜测
            "message": f"我猜你想的词是：{guess_word}。对吗？"
        }
```

**改进**：

- ✅ 系统在收集信息后进行决策
- ✅ 系统的决策被添加到 history
- ✅ 根据决策选择不同的 phase
- ✅ 用户知道接下来应该做什么

### `_handle_user_answer()` 对比

#### ❌ 修复前（错误）

```python
async def _handle_user_answer(self, game: dict, user_input: str, history: list[dict], intent_result: dict | None = None):
    # 1. 解析用户回答
    resolved_answer = self._resolve_user_answer(user_input, intent_result)

    # 2. 建立 history
    raw_history = history + [
        self._make_history_item(
            actor="user",
            turn_type="answer",
            input_text=user_input,
            answer_text=resolved_answer,
        )
    ]

    # 3. 直接返回 user_turn（错误！系统不进行决策）
    await self.repository.update_game_state(
        game["game_id"],
        history=raw_history,
        phase="user_turn",  # ← 回到用户轮
        next_actor="user",
        expected_turn_type="question",
    )

    return {
        "phase": "user_turn",
        "message": "回答已记录，现在轮到你提问或猜词。"
    }
```

**问题**：

- 用户回答后立即回到 user_turn
- 系统没有根据新信息做出决策
- 无法实现"回答→系统继续提问或猜测"的流程

#### ✅ 修复后（正确）

```python

    # 1. 解析用户回答
    resolved_answer = self._resolve_user_answer(user_input, intent_result)

    # 2. 建立 history
    raw_history = history + [
        self._make_history_item(
            actor="user",
            turn_type="answer",
            input_text=user_input,
            answer_text=resolved_answer,

    # 3. 系统进行决策（新增！）

        agent_question = agent_decision.get("question")
```

## 游戏流程对比

```

系统回答: "是的，属于理科"
返回 phase: user_turn ← ❌ 错误！

用户: "是物理吗？"
系统回答: "是的，属于物理"
返回 phase: user_turn ← ❌ 错误！

系统回答: "正确，就是量子力学"
返回 phase: user_turn ← ❌ 错误！

```

```
用户: "是理科吗？"
系统回答: "是的，属于理科"
```


## 轮次转移表对比

### ❌ 修复前（错误）

```
submit_turn()
  ├─ phase == "user_turn"
  │   ├─ intent == "question" → _handle_user_question()
  │   │   └─ → phase = "user_turn" ❌
  │   └─ intent == "guess" → _handle_user_guess()
  │       └─ → phase = "user_turn" (如果猜错) ❌
  │
  └─ phase == "awaiting_answer"
      ├─ intent == "answer" → _handle_user_answer()
      │   └─ → phase = "user_turn" ❌
      └─ ...
```

### ✅ 修复后（正确）

```
submit_turn()
  ├─ phase == "user_turn"
  │   ├─ intent == "question" → _handle_user_question()
  │   │   ├─ agent_decision.action == "question"
  │   │   │   └─ → phase = "awaiting_answer" ✅
  │   │   └─ agent_decision.action == "guess"
  │   │       └─ → phase = "awaiting_judgement" ✅
  │   └─ intent == "guess" → _handle_user_guess()
  │       └─ → phase = "user_turn" (猜错) ✅
  │
  └─ phase == "awaiting_answer"
      ├─ intent == "answer" → _handle_user_answer()
      │   ├─ agent_decision.action == "question"
      │   │   └─ → phase = "awaiting_answer" ✅
      │   └─ agent_decision.action == "guess"
      │       └─ → phase = "awaiting_judgement" ✅
      └─ ...
```

## 总结

修复的核心：**从"用户驱动的被动回答"转变为"系统主动决策的真正交替"**

- ❌ 之前：用户问 → 系统答 → 用户问 → 系统答 ...（无限循环）
- ✅ 之后：用户问 → 系统答+决策 → 用户回答/判断 → 系统决策 → ...（有序进行）
